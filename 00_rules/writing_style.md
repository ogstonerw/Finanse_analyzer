# Стиль текста диплома

- Использовать научный русский стиль.
- Избегать разговорных выражений, канцелярита и пустых фраз.
- Каждый подраздел должен:
  1. вводить тему,
  2. раскрывать ее применительно к задаче диплома,
  3. завершаться кратким выводом.
- Не использовать общие рассуждения без связи с платформой.
- Делать акцент на:
  - фондовом рынке,
  - новостных факторах,
  - макроэкономических данных,
  - методах ИИ,
  - облачной архитектуре,
  - прогнозировании реакции рынка.
- Не делать слишком длинные абзацы.
- По возможности использовать таблицы, схемы и авторские формулировки.

# Политика источников

## Основной приоритет
1. Официальные API и сайты
2. Биржевые и государственные источники
3. Disclosure-центры
4. Авторитетные деловые СМИ

## Обязательные категории источников
- рыночные данные
- макроэкономические данные
- корпоративные события
- новостной поток
- сырьевые данные

## Требования к источнику
- актуальность
- стабильность
- прозрачность происхождения данных
- применимость к проекту
- возможность регулярного обновления

## Реестр
- Все одобренные источники заносятся в 04_materials/sources_registry.md.
- Для каждого источника сохраняется дата последней проверки.

# Главное ТЗ для Codex по проекту diploma-market-ai

## Контекст проекта
Проект предназначен для подготовки бакалаврской выпускной квалификационной работы и разработки программного продукта.

Тема ВКР:
«Разработка облачной платформы анализа и прогнозирования реакции фондового рынка на основе информации из открытых источников с использованием методов искусственного интеллекта».

## Главная цель проекта
Разработать веб-платформу, которая:
- собирает открытые новостные, макроэкономические и рыночные данные;
- анализирует события, влияющие на рынок;
- рассчитывает технические и рыночные признаки;
- формирует недельный комбинированный прогноз реакции рынка;
- отображает индекс состояния рынка в виде кризисометра.

## Предметная область
Платформа ориентирована на:
- индекс Московской биржи;
- акции компаний:
  - Сбер,
  - Лукойл,
  - Газпром,
  - Яндекс;
- внешние сырьевые факторы:
  - нефть,
  - газ.

## Горизонт прогноза
Основной горизонт прогнозирования: 1 неделя.

## Результат прогноза
Система должна выдавать комбинированный сигнал, включающий:
- направление движения;
- силу сигнала;
- уверенность;
- краткое текстовое объяснение;
- связь сигнала с типом события и рыночным контекстом.

## Основные компоненты системы
1. Модуль сбора данных.
2. Модуль предобработки новостей и событий.
3. Модуль расчета технических признаков.
4. Интеллектуальный модуль анализа новостей и генерации сигнала.
5. Модуль расчета кризисометра.
6. Backend API.
7. Frontend на React.
8. Модуль авторизации пользователей по email и паролю.
9. Хранилище истории прогнозов.

## Технологический стек
- Backend: Go
- Frontend: React
- База данных: PostgreSQL
- AI/NLP: внешний API OpenAI
- Хранение истории прогнозов: обязательно
- Авторизация: email + password

## Логика прогнозирования
Сигнал не должен строиться исключительно на тексте новости.
Перед обращением к AI необходимо формировать и учитывать признаки:
- недельный тренд;
- доходность;
- RSI;
- волатильность;
- положение в канале;
- локальные уровни;
- тип события;
- общий режим рынка;
- влияние нефти и газа;
- макроэкономический контекст.

AI должен использоваться для:
- анализа текста новости;
- определения тональности;
- извлечения события;
- объединения текстовых, рыночных и технических признаков;
- генерации итогового комбинированного прогноза;
- формирования краткого объяснения прогноза.

## Кризисометр
Кризисометр должен быть гибридным интегральным индексом состояния рынка.
Он должен учитывать:
- рыночный стресс;
- новостный стресс;
- макроэкономический стресс;
- сырьевой фон;
- ширину рынка и поведение ключевых активов.

## Требования к структуре ВКР
Использовать структуру ВКР, согласованную с научным руководителем, даже если она отличается от типовых методических рекомендаций.

Структура:
Глава 1. Анализ предметной области и автоматизации
1.1. Особенности фондового рынка как объекта интеллектуального анализа
1.2. Классификация новостных и макроэкономических факторов, влияющих на рынок
1.3. Методы анализа тональности, извлечения событий и прогнозирования на финансовых данных
1.4. Анализ существующих платформ и инструментов оценки влияния новостных событий на фондовый рынок
1.5. Выработка рекомендаций к разработке облачной платформы анализа фондового рынка

Глава 2. Проектирование архитектуры и алгоритмов платформы
2.1. Структурная схема и общая архитектура облачной платформы
2.2. Проектирование базы данных и оптимизации структуры обработки данных
2.3. Проектирование интеллектуального модуля анализа новостей и генерации сигналов
2.4. Разработка пользовательского интерфейса и написание кода программы

## Работа с источниками
Перед разработкой модуля:
1. Проверять актуальные официальные источники данных.
2. Приоритет отдавать официальным API, биржам, регуляторам, статистическим службам и центрам раскрытия информации.
3. Все найденные релевантные источники добавлять в `04_materials/sources_registry.md`.
4. Для каждого источника фиксировать:
   - название,
   - URL,
   - тип данных,
   - способ доступа,
   - периодичность обновления,
   - статус,
   - дату последней проверки.
5. Не использовать сомнительные неофициальные источники как единственный источник.

# Стартовый реестр источников данных

## Назначение реестра

Данный реестр фиксирует стартовый набор источников данных для облачной платформы анализа и прогнозирования реакции фондового рынка. В реестр включены источники, соответствующие политике проекта: приоритет отдается официальным API, биржевым и государственным ресурсам, центрам раскрытия информации и только затем авторитетным деловым СМИ.

Дата последней проверки по стартовому реестру: `2026-04-07`.

## Структура записей

Для каждого источника фиксируются:

- название;
- URL;
- тип данных;
- способ доступа;
- периодичность обновления;
- статус;
- комментарий по применимости.

## Приоритет использования источников

При практической реализации модулей сбора данных рекомендуется придерживаться следующего порядка:

1. Официальные и биржевые источники для котировок, индексов, ставки ЦБ и макростатистики.
2. Центр раскрытия корпоративной информации для событий эмитентов.
3. Авторитетные деловые СМИ для расширения новостного покрытия и подтверждения значимости событий.

## Ограничения стартового реестра

- Реестр является стартовым и не исчерпывает все возможные источники проекта.
- Для части источников с программным доступом статус оставлен как `planned`, поскольку условия доступа и ограничения необходимо уточнить на этапе реализации коннекторов.
- Неофициальные источники в реестр не включались как основа для единственного канала данных.
